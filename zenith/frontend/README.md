# Zenith Frontend

Modern React frontend for Zenith AI Personal Assistant with glassmorphism design.

## Development

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## Environment Variables

Create a `.env` file:

```env
VITE_API_URL=http://localhost:8000
```

## Features

- ✨ Beautiful glassmorphism UI
- 🌓 Dark/Light theme support
- 🎙️ Voice assistant integration
- 💬 Real-time chat with streaming
- 📅 Calendar integration
- ✅ Tasks management
- 📝 Notes system
- 📧 Gmail integration
- 🎨 Animated mesh gradient background
- 📱 Fully responsive

## Tech Stack

- React 18
- TypeScript
- Vite
- Tailwind CSS
- Framer Motion
- React Query
- Lucide Icons
- React Markdown
- KaTeX (math rendering)
